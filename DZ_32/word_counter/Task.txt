Напишите функцию на PHP, которая принимает строку в качестве аргумента и подсчитывает частоту встречаемости каждого слова в этой строке. Функция должна возвращать ассоциативный массив, где ключами будут слова, а значениями - их частота встречаемости.

Пример использования:
$text = "The quick brown fox jumps over the lazy dog";
$result = countWordFrequency($text);
print_r($result);

Array
(
    [The] => 1
    [quick] => 1
    [brown] => 1
    [fox] => 1
    [jumps] => 1
    [over] => 1
    [the] => 1
    [lazy] => 1
    [dog] => 1
)


Ваша задача - реализовать функцию countWordFrequency(), которая будет принимать строку и возвращать ассоциативный массив с частотой встречаемости слов.
